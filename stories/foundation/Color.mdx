# Color

Color tokens define the visual identity across all BDS themes. The system separates primitive color scales from semantic color roles. Switch themes in the toolbar to see values update in real time.

## Department colors

Semantic tokens for categorical color-coding by organizational department (Clinical, Front Desk, Administration, etc.). Consumed via the `departmentColor(key)` helper in `src/lib/tokens.ts`. Valid keys: `blue`, `green`, `red`, `gold`, `purple`, `taupe`.

These tokens are **client-overridable** — override the full set in your theme file to match your client's departmental palette.
