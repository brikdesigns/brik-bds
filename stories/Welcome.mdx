# Brik Design System

The BDS component library provides themed React components powered by CSS custom properties. All components adapt automatically across 9 bundled themes.

## Theme switching

Use the paintbrush icon in the toolbar above to switch between themes. Every component and documentation page updates in real time — including the sidebar when switching to dark themes.

## Token system

Components use CSS variables from the Webflow export. The naming convention is:

```
--_[category]---[type]--[variant]
```

**Primitives** (raw scale values, never in components):
- `--space--400` — 16px
- `--font-size--700` — 32px
- `--grayscale--darkest` — #333

**Semantic** (purpose-bound, used in components):
- `--_color---background--brand-primary` — brand background color
- `--_typography---font-family--heading` — heading font family
- `--_space---lg` — large spacing value
- `--_border-radius---button` — button corner radius
