## Token system

Components use CSS custom properties generated from Figma via Style Dictionary:

```
Primitives (raw scale)              Semantic (purpose-bound)
──────────────────────              ────────────────────────
--space-400         → 16px          --padding-md   → var(--space-400)
--font-size-700     → 32px          --heading-lg   → var(--font-size-700)
--color-grayscale-darkest → #333    --text-primary → var(--color-grayscale-darkest)
```

**Components use semantic tokens only.** Primitives define the scale — never reference them directly.
