# Brik Design System

The live component playground. Every story renders the actual component with interactive controls, theme switching, and Chromatic baselines.

Narrative docs, usage guides, and API reference live at **[design.brikdesigns.com](https://design.brikdesigns.com)**.

---

## Sidebar

| Section | What's inside |
|---|---|
| **Components** | Atomic UI — buttons, inputs, badges, indicators, overlays |
| **Containers** | Bounded compositions — cards, sheets, tables, dialogs, forms |
| **Blocks** | Slot + atom combos — field, field-grid, date/time pickers, lists |
| **Layouts** | Arrangement primitives — stack, cluster, grid, frame |
| **Sections** | Page regions — nav, footer, breadcrumb, blueprints |
| **Foundation** | Tokens + atmospheres |
| **Content System** | Industries, voices, vocabularies |
| **Tools** | Theme switcher, health dashboard, contrast audit, token coverage |
