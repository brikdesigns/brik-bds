The five effect categories the Motion system covers. Each has reference implementations in `css/animations.css` (Lightweight), `css/premium-effects.css` (Premium CSS), or `js/premium-scroll.js` (Premium JS).

## Hover

Micro-interactions on user pointer entry. Lightweight tier ‚Äî pure CSS, no JS dependency.

- **`.hover-lift`** ‚Äî translateY -4px + shadow boost on hover.
- **`.hover-scale`** ‚Äî `transform: scale(1.02)` with `--duration-fast` + `--ease-out`.
- **`.hover-magnetic`** ‚Äî Premium tier; cursor-tracked translate driven by `js/premium-scroll.js`.
- **`.hover-color-shift`** ‚Äî color transition on `--text-brand-primary`.

All hover effects honor `prefers-reduced-motion: reduce` ‚Äî the transition resolves to `none`.

## Scroll Reveals

Element appears as it scrolls into the viewport. The `.is-visible` class is set by an IntersectionObserver (lightweight) or by GSAP ScrollTrigger (Premium). The CSS rules for the resting + revealed state live in `css/animations.css`.

- **`.animate-fade-up`** ‚Äî opacity 0 ‚Üí 1 + translateY 24px ‚Üí 0.
- **`.animate-fade-in`** ‚Äî opacity only.
- **`.animate-clip-reveal-up`** ‚Äî clip-path reveal from bottom edge.
- **`.animate-clip-reveal-diagonal`** ‚Äî clip-path reveal from a corner.

Stagger groups: wrap children with `.bds-stagger-1` through `.bds-stagger-6` to introduce a 100ms cascade between siblings.

## Overlays

Atmospheric layers ‚Äî grain, glass, vignettes. Mostly Premium-tier (CSS in `css/premium-effects.css`).

- **`.bds-grain`** ‚Äî SVG Perlin grain layer with configurable opacity. Used by `editorial-luxury` + `organic-textured` atmospheres.
- **`.bds-glass`** ‚Äî `backdrop-filter: blur(...) saturate(...)` panel.
- **`.bds-vignette`** ‚Äî radial gradient fade applied to a section background.
- **`.bds-spotlight`** ‚Äî cursor-tracked radial gradient. Section opt-in via `data-spotlight`.

## Typography

Text-specific motion ‚Äî gradient sweeps, stroke morphs, glow pulses, highlight reveals. Mix of Lightweight and Premium.

- **`.text-gradient-sweep`** ‚Äî animated gradient background-clip text.
- **`.text-stroke-fill`** ‚Äî stroke ‚Üí fill on scroll (Premium tier ‚Äî uses SVG mask).
- **`.text-highlight-reveal`** ‚Äî clip-path reveal of a highlight bar behind text.
- **`.text-letter-stagger`** ‚Äî Premium; letter-by-letter SplitText reveal (requires GSAP SplitText plugin).

## Video

Background and inline video patterns. Premium tier when scroll-scrubbed.

- **`.video-bg`** ‚Äî fullscreen background video with overlay token (`--background-overlay`).
- **`.video-bg-scrub`** ‚Äî Premium; video timeline driven by scroll position via `gsap.to(video, { currentTime: ... })`.
- **`.video-bg-loop-hero`** ‚Äî looping autoplay-muted hero pattern.
