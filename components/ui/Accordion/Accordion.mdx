# Accordion

Collapsible content sections for FAQs, settings panels, and grouped information. Each item has a title trigger and expandable content area with plus/minus icons.

## Variants

Behavior modes, pre-opened items, and rich content.

<Canvas of={Stories.Variants} />

- **Single open** — Only one item open at a time (default)
- **Multiple open** — `allowMultiple` allows simultaneous open items
- **Default open** — `defaultOpenItems` sets initially expanded items
- **Rich content** — Content supports any React node

## Usage

```tsx
import { Accordion } from '@brik/bds';

// Basic FAQ
<Accordion
  items={[
    { id: '1', title: 'Question one', content: 'Answer one' },
    { id: '2', title: 'Question two', content: 'Answer two' },
  ]}
/>

// Allow multiple open
<Accordion items={items} allowMultiple defaultOpenItems={['1', '3']} />

// Controlled
<Accordion items={items} openItems={openIds} onOpenChange={setOpenIds} />
```
