# Accordion

<ComponentLinks slug="accordion" />

Collapsible content sections for FAQs, settings panels, and grouped information. Each item has a title trigger and expandable content area with plus/minus icons.

## Playground

<Canvas of={Stories.Default} />

## Variants

Accordion has no visual-variant axis — behavior modes are Controls on the Playground:

- **Single open** — only one item open at a time (default)
- **Multiple open** — toggle `allowMultiple` for simultaneous open items
- **Default open** — set `defaultOpenItems` to pre-expand items on mount

Item `content` accepts any React node — ordered lists, multi-paragraph blocks — as shown below.
