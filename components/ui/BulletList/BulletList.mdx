# BulletList

Structured list of short text items — anti-messages, proof points, approved CTAs, brand rules. Replaces the raw `<ul style={{ margin: 0, paddingLeft, listStyleType }}>` pattern used across sheets.

Pass `marker="decimal"` for numbered lists; the underlying element switches to `<ol>` automatically.

## Usage

```tsx
import { BulletList, Field } from '@brikdesigns/bds';

<Field label="Anti-messages">
  <BulletList
    items={[
      'No price-first positioning',
      'No corporate-clinic language',
    ]}
  />
</Field>

// Numbered
<BulletList
  marker="decimal"
  items={['First', 'Second', 'Third']}
/>
```

## Rules

- **Use `items` prop, not `<li>` children.** Keeps call sites short and prevents accidental markup drift.
- **Don't use for long prose.** If items run multi-paragraph, you want `<BulletList>` nested inside a `SheetSection`, not standalone.
