# BulletList

<ComponentLinks slug="bullet-list" />

Structured list of short text items — anti-messages, proof points, approved CTAs, brand rules. Pass `marker="decimal"` for numbered lists; the element switches to `<ol>` automatically.

### Markers

Three marker styles — `disc` (default), `decimal`, `none`. Use the Controls panel on Default to try `density` (`compact` / `comfortable`).
