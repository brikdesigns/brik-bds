# Popover

Floating content panel anchored to a trigger element. Supports click or hover triggering, four placements, and controlled/uncontrolled modes.

## Usage

```tsx
import { Popover } from '@brik/bds';

// Click-triggered popover
<Popover content={<p>Additional details here</p>}>
  <button>More info</button>
</Popover>

// Hover-triggered, top placement
<Popover
  content={<p>Tooltip-style content</p>}
  trigger="hover"
  placement="top"
>
  <span>Hover me</span>
</Popover>
```
