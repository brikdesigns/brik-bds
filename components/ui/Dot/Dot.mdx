# Dot

Small status indicator circles that communicate state through color. Available in six status variants and three sizes.

---

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Statuses

<Canvas of={Stories.Statuses} />

- **`default`** -- Brand primary. General-purpose indicator.
- **`positive`** -- Green. Online, success, operational.
- **`warning`** -- Yellow. Away, attention needed.
- **`error`** -- Red. Offline, failure, outage.
- **`info`** -- Blue. Informational, in-progress.
- **`neutral`** -- Gray. Inactive, disabled.

## Usage

```tsx
import { Dot } from '@brik/bds';

// Basic status indicator
<Dot status="positive" />

// Different sizes
<Dot size="sm" status="warning" />
<Dot size="lg" status="error" />

// With label
<div style={{ display: 'flex', alignItems: 'center', gap: '8px' }}>
  <Dot status="positive" />
  <span>Online</span>
</div>
```
