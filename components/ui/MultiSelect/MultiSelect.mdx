# Multi-select

A select dropdown that supports multiple selections. Selected items appear as removable `Tag` chips below the dropdown. Composes `Select` + `Tag`.

## Usage

```tsx
import { MultiSelect } from '@brik/bds';

const [selected, setSelected] = useState<string[]>([]);

<MultiSelect
  label="Services"
  placeholder="Select services..."
  options={[
    { label: 'Brand Design', value: 'brand' },
    { label: 'Marketing', value: 'marketing' },
    { label: 'Product Design', value: 'product' },
  ]}
  value={selected}
  onChange={setSelected}
/>
```
