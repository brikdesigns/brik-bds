# SidebarNavigation

A fixed-position sidebar navigation component with logo, navigation links, footer actions, and user section. Uses BDS tokens for consistent theming.

## Features

- **Fixed Position**: Stays visible while content scrolls
- **BDS Themed**: Uses design tokens for consistent styling
- **Flexible Content**: Supports logo, nav items, footer actions, and user section
- **Active State**: Visual indicator for current page
- **Icons**: Optional icons for navigation items (any ReactNode)
- **Custom Width**: Configurable sidebar width

## Usage

### Basic

```tsx
import { SidebarNavigation } from '@/components/ui/SidebarNavigation';

<SidebarNavigation
  logo={<img src="/logo.svg" alt="Logo" />}
  navItems={[
    { label: 'Dashboard', href: '/dashboard', active: true },
    { label: 'Projects', href: '/projects' },
    { label: 'Settings', href: '/settings' },
  ]}
/>
```

### With Footer Actions and User Section

```tsx
<SidebarNavigation
  logo={<YourLogo />}
  navItems={navItems}
  footerActions={<Button>New Project</Button>}
  userSection={<UserInfo name="John Doe" />}
/>
```

## Props

### SidebarNavigation

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `logo` | `ReactNode` | required | Logo element to display at top |
| `navItems` | `SidebarNavItem[]` | required | Array of navigation items |
| `footerActions` | `ReactNode` | - | Footer content (buttons, links, etc.) |
| `userSection` | `ReactNode` | - | User info section at bottom |
| `width` | `string` | `"260px"` | Width of sidebar (CSS value) |

### SidebarNavItem

| Prop | Type | Description |
|------|------|-------------|
| `label` | `string` | Display label for the navigation item |
| `href` | `string` | Link destination |
| `active` | `boolean` | Whether this item is currently active (optional) |
| `icon` | `ReactNode` | Optional icon element (emoji, SVG, etc.) |

## Design Tokens

The component uses the following BDS tokens:

**Colors:**
- `--_color---surface--primary` - Sidebar background
- `--_color---border--secondary` - Borders
- `--_color---text--primary` - Active item text
- `--_color---text--secondary` - Default item text
- `--_color---page--secondary` - Active item background
- `--brand--primary` - Active indicator dot

**Typography:**
- `--_typography---font-family--body` - Navigation font

**Layout:**
- Fixed positioning with customizable width (default: 260px)
- Responsive padding using pixel values
- Border radius: 6px for navigation items

## Sections

The sidebar has four sections:

1. **Logo section** (required) - Top section with your logo or branding
2. **Navigation section** (required) - List of navigation links
3. **Footer actions** (optional) - Bottom section for buttons or actions
4. **User section** (optional) - User info/avatar at the very bottom

## Navigation Items

Items without icons will show a colored dot indicator instead (brand color when active, gray when inactive).

## Implementation Notes

- The sidebar uses `position: fixed` - ensure your main content area accounts for the sidebar width
- Navigation items use `<a>` tags with `href` - integrate with your routing solution (e.g., Next.js Link)
- Active state is controlled per-item via the `active` boolean property
- Icons can be any React element (emoji, SVG, Font Awesome components, etc.)
- The sidebar includes scroll behavior for overflow navigation items
