# Sidebar navigation

<ComponentLinks slug="sidebar-navigation" />

Fixed-position sidebar with logo, navigation links, footer actions, and user section. Width is configurable via prop.

## Variants

Default, with icons, with user section, and custom width configurations.

- **`logo`** (required) — top section with branding
- **`navItems`** (required) — navigation links with active state
- **`footerActions`** — bottom actions (buttons, links)
- **`user`** — user info at very bottom
