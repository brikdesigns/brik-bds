# Sidebar navigation

Fixed-position sidebar with logo, navigation links, footer actions, and user section. Width is configurable via prop.

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Variants

Default, with icons, with user section, and custom width configurations.

<Canvas of={Stories.Variants} />

| Section | Required | Description |
|---------|----------|-------------|
| **Logo** | Yes | Top section with branding |
| **Nav items** | Yes | Navigation links with active state |
| **Footer** | No | Bottom actions (buttons, links) |
| **User** | No | User info at very bottom |

Full application layout with sidebar and main content area.

<ArgTypes of={Stories} />

## Usage

```tsx
import { SidebarNavigation } from '@brik/bds';
import { Button } from '@brik/bds';

<SidebarNavigation
  logo={<img src="/logo.svg" alt="Logo" />}
  navItems={[
    { label: 'Dashboard', href: '/dashboard', active: true },
    { label: 'Projects', href: '/projects' },
    { label: 'Settings', href: '/settings' },
  ]}
  footerActions={<Button variant="primary" size="sm" fullWidth>New Project</Button>}
  width="260px"
/>
```
