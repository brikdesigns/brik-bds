# Sidebar navigation

<ComponentLinks slug="sidebar-navigation" />

Fixed-position primary sidebar with logo, nav items (`NavItem` internally), footer slot, and profile slot. Toggle `collapsed` for icon-only mode — pair with `SubNavigation` for a two-column app shell.

## Playground

Toggle `collapsed`, `showProfile`, `showFooterActions`, or override `width` in the Controls panel to explore states.
