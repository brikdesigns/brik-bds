# Icons

Iconify + Phosphor icon reference for BDS. Icons render as inline SVGs via `@iconify/react` using the [Phosphor](https://phosphoricons.com/) collection (`ph:*`).

---

## Setup

| Package | Purpose |
|---------|---------|
| `@iconify/react` | Renders any Iconify icon as a React `<Icon>` component |
| `@iconify-json/ph` | Phosphor icon data (bundled for offline/SSR use) |

BDS registers the Phosphor collection at startup in `.storybook/preview.tsx` via `addCollection()`. No CDN calls needed.

### Usage

```tsx
import { Icon } from '@iconify/react';

// Direct string reference
<Icon icon="ph:star-fill" />

// Or import from BDS constants (preferred inside BDS components)
import { Star } from '../../icons';
<Icon icon={Star} />
```

BDS icon constants live in `components/icons.ts` and follow the pattern `export const Name = 'ph:icon-name';`.

## Adding icons

1. Find the icon name at [phosphoricons.com](https://phosphoricons.com/)
2. Add a named constant to `components/icons.ts` using the `ph:icon-name` format
3. Add to the relevant category story above
4. If a new category is needed, add a new story section

## Variants

Phosphor provides six weights per icon: `ph:name` (regular), `ph:name-bold`, `ph:name-fill`, `ph:name-light`, `ph:name-thin`, `ph:name-duotone`. BDS uses regular and fill variants.
