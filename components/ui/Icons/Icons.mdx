# Icons

Font Awesome 7 icon reference for BDS. This is a **living document** — add new icons here as they're adopted in components.

---

## Setup

| Package | Version | Style |
|---------|---------|-------|
| `@fortawesome/free-solid-svg-icons` | 7.2+ | Solid (filled) |
| `@fortawesome/react-fontawesome` | 3.2+ | React wrapper |

Storybook configures `autoAddCss = false` in `.storybook/preview.tsx` — icon styles are imported once via `@fortawesome/fontawesome-svg-core/styles.css`.

## Adding icons

1. Import from `@fortawesome/free-solid-svg-icons`
2. Add to the relevant category story above
3. If a new category is needed, add a new story section

## Outline vs filled

Only **solid** (filled) is installed. To add outline style:

```bash
npm i @fortawesome/free-regular-svg-icons
```

Then import from `@fortawesome/free-regular-svg-icons` with the `far` prefix convention.
