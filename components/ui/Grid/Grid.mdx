# Grid

<ComponentLinks slug="grid" />

Grid arranges children in a CSS grid. Set a fixed `columns` count (1–6), or `auto-fit` / `auto-fill` (default `auto-fit`) to reflow items to fill the available width; `gap` uses the spacing scale.
