# Empty state

Placeholder for screens with no data. Use when a list, table, or page has no content to display, guiding users toward their next action.

---

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Content shapes

The four meaningful states an empty state can take. Pick the shape that matches the recovery context: a CTA when there's an action to take, description-only when the user just needs to adjust filters, title-only when even a description would be noise, custom children when the slot needs an illustration or a drop zone.

## Usage

```tsx
import { EmptyState } from '@brik/bds';

// With action button
<EmptyState
  title="No projects yet"
  description="Create your first project to get started."
  buttonProps={{ children: 'Create Project', onClick: handleCreate }}
/>

// Informational only
<EmptyState
  title="No results found"
  description="Try adjusting your search or filters."
/>
```
