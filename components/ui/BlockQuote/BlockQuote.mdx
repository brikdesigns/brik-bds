# BlockQuote

<ComponentLinks slug="block-quote" />

Inline styled block-quote — `<figure>/<blockquote>/<figcaption>/<cite>` — for narrative body content like customer stories and blog posts. Optionally accents to a service line's color. Distinct from `Testimonial`, which is a discrete card with avatar / rating slots; BlockQuote flows within a longer piece.
