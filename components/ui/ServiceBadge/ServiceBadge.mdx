# Service Badge

Colored badge identifying Brik service categories. Each category has a dedicated color palette used across the Brik ecosystem for consistent service identification.

| **Information** | Blue | `--services--blue` |
| **Product** | Purple | `--services--purple` |
| **Back Office** | Orange | `--services--orange` |

## Usage

```tsx
import { ServiceBadge } from '@brik/bds';

// Category badge
<ServiceBadge category="marketing" />

// With size
<ServiceBadge category="brand" size="lg" />

// Label mode
<ServiceBadge category="product" mode="label" />

// With specific service icon
<ServiceBadge category="service" serviceName="CRM Setup and Data Cleanup" />
```
