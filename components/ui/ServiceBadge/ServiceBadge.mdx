# Service Badge

Colored badge identifying Brik service categories. Each category has a dedicated color palette (primary, dark, light) used across the Brik ecosystem for consistent service identification.

## Usage

```tsx
import { ServiceBadge } from '@brik/bds';

// Category badge
<ServiceBadge category="marketing" />

// With size
<ServiceBadge category="brand" size="lg" />

// Label mode
<ServiceBadge category="product" mode="label" />

// With specific service icon (back office category)
<ServiceBadge category="service" serviceName="CRM Setup and Data Cleanup" />
```

## Token reference

| Category | Primary | Dark | Light |
|---|---|---|---|
| Marketing | `--services--green` | `--services--green-dark` | `--services--green-light` |
| Brand | `--services--yellow` | `--services--yellow-dark` | `--services--yellow-light` |
| Information | `--services--blue` | `--services--blue-dark` | `--services--blue-light` |
| Product | `--services--purple` | `--services--purple-dark` | `--services--purple-light` |
| Back Office | `--services--orange` | `--services--orange-dark` | `--services--orange-light` |
