# CatalogPicker

<ComponentLinks slug="catalog-picker" />

Industry-aware multi-pick input. Renders a reference catalog as suggestion-backed picks and normalizes every entry with a stable slug plus `source: 'catalog' | 'custom'` attribution.

## Default

<Canvas of={Stories.Default} />

## Variants

Empty state, `strict` (catalog-only), and swapping the `catalog` for another industry are all Controls / code props on any story. This shows source attribution — the one behavior the output array makes visible.

### Mixed sources

Each entry is tagged by origin — catalog picks and custom additions live together in the same output array.
