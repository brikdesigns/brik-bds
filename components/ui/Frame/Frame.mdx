# Frame

<ComponentLinks slug="frame" />

Frame locks a child to a fixed aspect ratio. `ratio` picks a named preset (`4-3` default, `square`, `cinema`, …) or `customRatio` takes any CSS `aspect-ratio` string; `fit` controls how the child fills the box — `cover` crops, `contain` letterboxes.
