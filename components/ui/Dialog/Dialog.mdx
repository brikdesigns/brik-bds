# Dialog — deprecated

> **This component is deprecated.** Use `<Modal preset="confirm">` instead. Same behavior, same API (`title`, `description`, `confirmLabel`, `cancelLabel`, `onConfirm`), with `confirmVariant="destructive"` covering this component's `variant="destructive"`.

This page exists for migration reference only. New code should not import `Dialog`.

---

## Migration

```tsx
// Before
<Dialog
  isOpen={open}
  onClose={close}
  title="Delete item?"
  description="This cannot be undone."
  variant="destructive"
  confirmLabel="Delete"
  onConfirm={handleDelete}
/>

// After
<Modal
  preset="confirm"
  isOpen={open}
  onClose={close}
  title="Delete item?"
  description="This cannot be undone."
  confirmVariant="destructive"
  confirmLabel="Delete"
  onConfirm={handleDelete}
/>
```
