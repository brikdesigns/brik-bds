# Avatar

User profile images with automatic fallback to initials. Supports multiple sizes and optional status indicators for presence states.

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Variants

Sizes, content types (initials vs image), and status indicators.

<Canvas of={Stories.Variants} />

| Size | Dimensions | Use for |
|------|-----------|---------|
| **sm** | 32px | Compact lists, inline mentions |
| **md** | 40px | Default, cards, comments |
| **lg** | 48px | Profile headers, featured users |
| **xl** | 64px | Hero profiles, account settings |

## Usage

```tsx
import { Avatar } from '@brik/bds';

// Initials fallback
<Avatar name="John Doe" />

// With image
<Avatar src="/user.jpg" alt="User avatar" name="John Doe" />

// With status
<Avatar name="Jane Smith" status="online" />

// Different sizes
<Avatar name="Bob Johnson" size="lg" />
```
