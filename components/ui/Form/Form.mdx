# Form

Structured form container with layout control, error/success messaging, and consistent spacing. Wraps form fields with title, description, and footer actions.

## Layouts

<Canvas of={Stories.Vertical} />
<Canvas of={Stories.Horizontal} />

- **`vertical`** — Fields stack vertically (default)
- **`horizontal`** — Fields side by side for compact filter/search forms

Gap sizes:

- **`sm`** — `--gap-md` (8px)
- **`md`** — `--gap-lg` (16px)
- **`lg`** — `--gap-xl` (24px)

## Usage

```tsx
import { Form, TextInput, TextArea, Button } from '@brik/bds';

<Form
  title="Contact Us"
  description="We'll get back to you within 24 hours."
  onSubmit={handleSubmit}
  footer={<Button type="submit">Send Message</Button>}
>
  <TextInput label="Name" placeholder="Your name" fullWidth />
  <TextInput label="Email" type="email" placeholder="you@example.com" fullWidth />
  <TextArea label="Message" placeholder="How can we help?" fullWidth />
</Form>
```
