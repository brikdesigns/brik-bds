# Pagination

Page navigation controls with previous/next arrows, page numbers, and ellipsis for large page counts. Supports left, center, and right alignment.

## Usage

```tsx
import { Pagination } from '@brik/bds';

const [page, setPage] = useState(1);

<Pagination
  currentPage={page}
  totalPages={80}
  onChange={setPage}
  position="center"
/>
```
