# Progress dots

Minimal step indicator using dots. Use for onboarding flows, carousels, and wizards where step labels aren't needed.

## Usage

```tsx
import { ProgressDots } from '@brik/bds';

// Basic progress dots
<ProgressDots count={5} activeStep={2} />

// Small, linear mode
<ProgressDots count={4} activeStep={0} size="sm" linear />

// With step change handler
<ProgressDots
  count={3}
  activeStep={currentStep}
  onDotClick={(step) => setCurrentStep(step)}
/>
```
