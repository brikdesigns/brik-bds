# Collapsible card

Expandable content section with a clickable header. Use for FAQs, settings panels, and long-form content that benefits from progressive disclosure.

## Overview

<Canvas of={Stories.Default} />

## States

### Collapsed (default)
Content is hidden, only the header is visible.
<Canvas of={Stories.Default} />

### Expanded
Content is visible below the header.
<Canvas of={Stories.Open} />

## Variations

### Without section label
Omit the section label for simpler layouts.
<Canvas of={Stories.NoSectionLabel} />

### With rich content
Supports any React content inside the expandable area.
<Canvas of={Stories.WithRichContent} />

## Multiple sections

Stack multiple collapsible cards for accordion-style layouts.
<Canvas of={Stories.MultipleSections} />

## Props

<ArgTypes of={Stories} />

## Usage

```tsx
import { CollapsibleCard } from '@brik/bds';

// Basic collapsible card
<CollapsibleCard
  sectionLabel="Section 01"
  title="Overview and Goals"
>
  <p>Content goes here...</p>
</CollapsibleCard>

// Initially open
<CollapsibleCard
  title="Settings"
  defaultOpen
>
  <p>Settings content...</p>
</CollapsibleCard>
```
