# Button

Triggers an event or action. The button family includes three components for different semantic needs.

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Variants and sizes

Seven variants across three sizes. Standard variants for general actions, danger variants for destructive actions.

<Canvas of={Stories.Variants} />

| Variant | Use for |
|---------|---------|
| **primary** | Main call-to-action. One per section. |
| **outline** | Secondary emphasis. Alternative to primary. |
| **secondary** | Tertiary/subtle. Supporting actions. |
| **ghost** | Minimal emphasis. Navigation, repeated actions. |
| **danger** | Destructive primary action (delete, remove). |
| **danger-outline** | Destructive with less emphasis. |
| **danger-ghost** | Destructive, minimal emphasis (inline delete). |

## Icons

Use `iconBefore` and `iconAfter` props. Icons should use `1em` sizing to scale with the button.

<Canvas of={Stories.Icons} />

## Interactive states

Hover, focus, active, disabled, and loading states. Each variant has custom hover behavior.

<Canvas of={Stories.States} />

## Loading

Spinner replaces content while preserving button width. Button is disabled during loading.

<Canvas of={Stories.Loading} />

## LinkButton

An `<a>` element styled as a button. Use when the action navigates to a URL. The `href` prop is required.

<Canvas of={Stories.Link} />

```tsx
import { LinkButton } from '@brik/bds';

<LinkButton href="/docs" variant="outline">Read docs</LinkButton>
```

## IconButton

An icon-only button with a required `label` prop for accessibility.

<Canvas of={Stories.IconOnly} />

```tsx
import { IconButton } from '@brik/bds';

<IconButton icon={<CloseIcon />} label="Close dialog" variant="ghost" />
<IconButton icon={<TrashIcon />} label="Delete item" variant="danger-ghost" />
```

## Props

### Button

<ArgTypes of={Stories} />

### LinkButton

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | *required* | Link destination |
| `variant` | `ButtonVariant` | `'primary'` | Visual style variant |
| `size` | `ButtonSize` | `'md'` | Button size |
| `fullWidth` | `boolean` | `false` | Full width |
| `iconBefore` | `ReactNode` | — | Icon before text |
| `iconAfter` | `ReactNode` | — | Icon after text |

### IconButton

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `ReactNode` | *required* | The icon to render |
| `label` | `string` | *required* | Accessible label (aria-label) |
| `variant` | `ButtonVariant` | `'ghost'` | Visual style variant |
| `size` | `ButtonSize` | `'md'` | Button size |
| `loading` | `boolean` | `false` | Loading state |

## Usage

```tsx
import { Button, LinkButton, IconButton } from '@brik/bds';

// Standard action
<Button variant="primary" size="md">Save changes</Button>

// Navigation
<LinkButton href="/docs" variant="outline">Documentation</LinkButton>

// Icon-only
<IconButton icon={<EditIcon />} label="Edit" variant="secondary" />

// Destructive action
<Button variant="danger">Delete project</Button>
```
