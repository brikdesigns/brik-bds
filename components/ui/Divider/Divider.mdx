# Divider

A visual separator used to divide content sections. Supports horizontal and vertical orientations with configurable spacing.

## Overview

Dividers can include built-in margin to create breathing room between content sections.

## Usage

```tsx
import { Divider } from '@brik/bds';

// Basic horizontal divider
<Divider />

// With spacing
<Divider spacing="lg" />

// Vertical in a flex row
<div style={{ display: 'flex', alignItems: 'center' }}>
  <span>Left</span>
  <Divider orientation="vertical" spacing="md" />
  <span>Right</span>
</div>
```
