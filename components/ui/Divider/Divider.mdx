## Variants

Orientations and spacing options.

<Canvas of={Stories.Variants} />

- **`none`** -- 0px. Tight separators, table rows.
- **`sm`** -- 12px. Compact spacing.
- **`md`** -- 16px. Default breathing room.
- **`lg`** -- 24px. Section-level separation.
