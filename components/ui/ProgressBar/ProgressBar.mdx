# Progress bar

Visual indicator showing completion progress. Use for file uploads, multi-step forms, loading states, and any task with measurable progress.

## Usage

```tsx
import { ProgressBar } from '@brik/bds';

<ProgressBar value={35} label="Upload progress" />
```

## Accessibility

The component uses `role="progressbar"` with `aria-valuenow`, `aria-valuemin`, and `aria-valuemax`. Provide a `label` prop for screen reader context.
