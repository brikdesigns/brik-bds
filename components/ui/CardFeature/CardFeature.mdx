# Card feature

Highlights a single feature or benefit with an icon, title, and description. Use in feature grids, landing pages, and product overviews.

## Variants

Left and center alignment, with action link, and without icon.

<Canvas of={Stories.Variants} />

- **`left`** — Icon and text start-aligned (default)
- **`center`** — Center-aligned for symmetrical grids

## Usage

```tsx
import { CardFeature } from '@brik/bds';
import { Icon } from '@iconify/react';

<CardFeature
  icon={<Icon icon="ph:lightning" />}
  title="Fast Performance"
  description="Optimized for speed and reliability."
/>
```
