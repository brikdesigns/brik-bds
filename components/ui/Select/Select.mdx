# Select

Dropdown selection control with options array, placeholder text, optional leading icon, and size variants. Use for choosing from a predefined set of values in forms and filters.

## Overview

<Canvas of={Stories.Default} />

## Width behavior

The select fills its container by default (`fullWidth` is true). This matches standard form field behavior — the select adapts fluidly to whatever container it lives in.

- **Full width (default)** — fills parent container, responsive to viewport
- **Constrained** — respects `max-width` on a parent wrapper
- **Compact** — set `fullWidth={false}` for inline usage where the select shrinks to content

## Interaction states

All states are handled via CSS pseudo-selectors in `Select.css`:

| State | Visual | Token |
|-------|--------|-------|
| Default | Light border | `--border-input` |
| Hover | Darker border | `--border-primary` |
| Focus | Brand border + ring | `--border-brand-primary` |
| Error | Red border + ring | `--color-system-red` |
| Disabled | 50% opacity | — |

## Usage

```tsx
import { Select } from '@brik/bds';
import { FontAwesomeIcon } from '@fortawesome/react-fontawesome';
import { faBuilding, faGlobe } from '@fortawesome/free-solid-svg-icons';

// Basic select (fills container by default)
<Select
  placeholder="Select one..."
  options={[
    { label: 'First', value: 'first' },
    { label: 'Second', value: 'second' },
  ]}
/>

// With icon and label
<Select
  label="Company"
  placeholder="Select company..."
  options={options}
  icon={<FontAwesomeIcon icon={faBuilding} />}
/>

// Grouped options
<Select
  label="Location"
  placeholder="Select city..."
  options={[
    {
      label: 'United States',
      options: [
        { label: 'New York', value: 'ny' },
        { label: 'Chicago', value: 'chi' },
      ],
    },
    {
      label: 'Europe',
      options: [
        { label: 'London', value: 'lon' },
        { label: 'Paris', value: 'par' },
      ],
    },
  ]}
  icon={<FontAwesomeIcon icon={faGlobe} />}
/>

// Compact inline usage
<Select
  size="sm"
  placeholder="Filter..."
  options={filters}
  fullWidth={false}
/>
```
