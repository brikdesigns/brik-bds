# Toast

White surface notification with optional colored Badge icon indicating status. The surface never changes color — only the badge communicates success, error, warning, or info.

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Variants

All status variants, title-only, and non-dismissible configurations.

<Canvas of={Stories.Variants} />

| Variant | Badge | Icon |
|---------|-------|------|
| **default** | None | — |
| **success** | Positive | circle-check |
| **error** | Error | circle-exclamation |
| **warning** | Warning | triangle-exclamation |
| **info** | Info | circle-info |

## Usage

```tsx
import { Toast } from '@brik/bds';

<Toast
  title="Changes saved"
  description="Your settings have been updated"
  variant="success"
  onDismiss={() => setShowToast(false)}
/>
```
