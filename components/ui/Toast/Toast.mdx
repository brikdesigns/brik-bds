# Toast

Dark overlay notification for transient feedback. Use for confirmations, status updates, and brief messages that don't require user action.

## Usage

```tsx
import { Toast } from '@brik/bds';

<Toast
  title="Changes saved"
  description="Your settings have been updated"
  onDismiss={() => setShowToast(false)}
/>
```

## Positioning

The Toast component does not manage its own position. Wrap it in a fixed/absolute container to position it on screen:

```tsx
<div style={{ position: 'fixed', bottom: 32, left: '50%', transform: 'translateX(-50%)' }}>
  <Toast title="Saved" onDismiss={handleDismiss} />
</div>
```
