# Breadcrumb

Navigation breadcrumb trail showing the current page location within a hierarchy. Supports slash and chevron separators.

## Usage

```tsx
import { Breadcrumb } from '@brik/bds';

<Breadcrumb
  items={[
    { label: 'Home', href: '/' },
    { label: 'Products', href: '/products' },
    { label: 'Design System' },
  ]}
  separator="chevron"
/>
```
