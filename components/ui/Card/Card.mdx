# Card

<ComponentLinks slug="card" />

Flexible content container for grouping related information. Composable with `CardTitle`, `CardDescription`, and `CardFooter`. Supports two preset shapes тАФ `control` and `summary` тАФ that subsume the legacy `CardControl` and `CardSummary` components.

## Variants

All visual variants, padding sizes, interactive, and link modes.

<Canvas of={Stories.Variants} />

- **`outlined`** тАФ Subtle secondary border (default)
- **`brand`** тАФ Primary-color border for emphasis
- **`elevated`** тАФ Shadow for visual prominence

### Control preset

`preset="control"` renders a settings/control row: leading badge + (title + description) on the left, action slot on the right.

<Canvas of={Stories.ControlPreset} />

### Summary preset

`preset="summary"` renders a compact metric layout: label on top, large value below, optional text link to the right. Numeric values format as locale integers; pass `type="price"` for USD currency.
