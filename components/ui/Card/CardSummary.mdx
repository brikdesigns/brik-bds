# Card summary — deprecated

> **This component is deprecated.** Use `<Card preset="summary">` instead. Same prop names (`label`, `value`, `type`, `textLink`), same layout, same number formatting.

This page exists for migration reference only. New code should not import `CardSummary`.

---

## Migration

```tsx
// Before
<CardSummary label="Total" value={42} type="numeric" />

// After
<Card preset="summary" label="Total" value={42} type="numeric" />
```
