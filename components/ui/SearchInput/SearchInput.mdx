# Search input

Text input with a built-in magnifying glass icon for search functionality. Supports medium and small sizes for different UI densities.

## Usage

```tsx
import { SearchInput } from '@brik/bds';

// Basic search
<SearchInput placeholder="Search..." />

// Small size
<SearchInput size="sm" placeholder="Quick search..." />

// With label, full width
<SearchInput label="Search" placeholder="Search products..." fullWidth />
```
