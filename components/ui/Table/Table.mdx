# Table

Data table with composable sub-components. Supports sorting, selection, striped rows, size variants, and custom cell content.

## Playground

Use the Controls panel to toggle `striped` and `size` props.

<Canvas of={Stories.Playground} />

## Variants

Size variants, striped rows, sortable headers, and all cell types: icon left/right, badge, tag, checkbox, data input, brand badge, actions, button, text link, and two-line layouts.

<Canvas of={Stories.Variants} />

| Size | Description |
|------|-------------|
| **default** | Compact padding for dense data |
| **comfortable** | Generous padding (72px cell height) |

### Size propagation

Size is set on `<Table>` via `data-size` attribute. CSS selectors apply padding to all child `<th>` and `<td>` elements automatically — no React context needed.

Full-featured composite table and striped comfortable layout.

## Sub-components

| Component | Element | Description |
|-----------|---------|-------------|
| `Table` | `<table>` | Container with `striped` and `size` props |
| `TableHeader` | `<thead>` | Header section wrapper |
| `TableBody` | `<tbody>` | Body section wrapper |
| `TableRow` | `<tr>` | Row with optional `selected` state |
| `TableHead` | `<th>` | Header cell with optional `sortable` / `sortDirection` |
| `TableCell` | `<td>` | Body cell for data |

## Usage

```tsx
import { Table, TableHeader, TableBody, TableRow, TableHead, TableCell } from '@brik/bds';
import { Badge } from '@brik/bds';

<Table striped size="comfortable">
  <TableHeader>
    <TableRow>
      <TableHead sortable sortDirection="asc">Name</TableHead>
      <TableHead>Email</TableHead>
      <TableHead>Status</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>Alice Chen</TableCell>
      <TableCell>alice@example.com</TableCell>
      <TableCell><Badge status="positive" size="sm">Active</Badge></TableCell>
    </TableRow>
  </TableBody>
</Table>
```
