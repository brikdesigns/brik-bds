# Table

Data table component with sub-components for structured content display. Supports sorting, selection, and custom cell content with 12 Figma-defined cell types.

## Overview

<Canvas of={Stories.Default} />

## Sub-components

The Table system is composable using these elements:
- `Table` — Container with optional striped prop
- `TableHeader` — Header section wrapper
- `TableBody` — Body section wrapper
- `TableRow` — Row container with optional selected state
- `TableHead` — Header cell with optional sortable props
- `TableCell` — Body cell for data

## Cell types

### Text cell
Plain text data in cells. The default cell type.

Text with a leading icon for visual context (24x24 icon wrapper).

## Usage

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableRow,
  TableHead,
  TableCell
} from '@brik/bds';
import { Badge } from '@brik/bds';
import { Tag } from '@brik/bds';
import { Button } from '@brik/bds';
import { Checkbox } from '@brik/bds';
import { Input } from '@brik/bds';
import { Link } from '@brik/bds';
import { FontAwesomeIcon } from '@fortawesome/react-fontawesome';
import { faPenToSquare, faTrash, faPalette } from '@fortawesome/free-solid-svg-icons';

// Text cell (default)
<TableCell>Alice Chen</TableCell>

// Text + icon left cell
<TableCell>
  <span style={{ display: 'flex', alignItems: 'center', gap: 4 }}>
    <FontAwesomeIcon icon={faPalette} style={{ width: 24 }} />
    Design
  </span>
</TableCell>

// Badge cell
<TableCell>
  <Badge status="positive" size="sm">Active</Badge>
</TableCell>

// Tag cell
<TableCell>
  <Tag size="sm">Design</Tag>
</TableCell>

// Checkbox cell
<TableCell>
  <Checkbox name="select" label="" value="alice" />
</TableCell>

// Data input cell
<TableCell>
  <Input size="sm" placeholder="Enter value" />
</TableCell>

// Icon action buttons
<TableCell style={{ textAlign: 'right' }}>
  <button style={iconButtonStyles} aria-label="Edit">
    <FontAwesomeIcon icon={faPenToSquare} />
  </button>
  <button style={iconButtonStyles} aria-label="Delete">
    <FontAwesomeIcon icon={faTrash} />
  </button>
</TableCell>

// Button cell
<TableCell>
  <Button variant="primary" size="sm">View profile</Button>
</TableCell>

// Text link cell
<TableCell>
  <Link href="#">View profile</Link>
</TableCell>

// 2-line text cell
<TableCell>
  <div>
    <div style={{ fontWeight: 600 }}>Alice Chen</div>
    <div style={{ fontSize: 'var(--_typography---body--sm)', color: 'var(--_color---text--muted)' }}>
      alice@example.com
    </div>
  </div>
</TableCell>
```
