# Table

Data table component with sub-components for structured content display. Supports sorting, selection, and custom cell content.

## Overview

<Canvas of={Stories.Default} />

## Sub-components

The Table system is composable using these elements:
- `Table` — Container with optional striped prop
- `TableHeader` — Header section wrapper
- `TableBody` — Body section wrapper
- `TableRow` — Row container with optional selected state
- `TableHead` — Header cell with optional sortable props
- `TableCell` — Body cell for data

## Usage

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableRow,
  TableHead,
  TableCell
} from '@brik/bds';

<Table>
  <TableHeader>
    <TableRow>
      <TableHead sortable sortDirection="asc">Name</TableHead>
      <TableHead>Email</TableHead>
      <TableHead>Role</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow selected>
      <TableCell>Alice Chen</TableCell>
      <TableCell>alice@example.com</TableCell>
      <TableCell>Admin</TableCell>
    </TableRow>
    <TableRow>
      <TableCell>Bob Smith</TableCell>
      <TableCell>bob@example.com</TableCell>
      <TableCell>Editor</TableCell>
    </TableRow>
  </TableBody>
</Table>
```
