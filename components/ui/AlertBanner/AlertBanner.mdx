# Alert banner

Contextual banner for important messages requiring user attention. Uses a secondary surface background with an icon-only Badge, title, description, and optional action.

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Variants

All alert variants with badge icons, plus title-only usage.

<Canvas of={Stories.Variants} />

| Variant | Badge status | Icon | Color |
|---------|-------------|------|-------|
| **information** | info | circle-info | Neutral gray |
| **warning** | warning | triangle-exclamation | Yellow |
| **error** | error | triangle-exclamation | Red |

## Patterns

Real-world usage: maintenance notices, error recovery, and informational alerts.

## Usage

```tsx
import { AlertBanner, Button } from '@brik/bds';

<AlertBanner
  variant="information"
  title="Update available"
  description="A new version is ready to install"
  action={<Button variant="primary" size="sm">Update now</Button>}
/>
```
