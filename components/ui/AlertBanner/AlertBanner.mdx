# Alert banner — deprecated

> **This component is deprecated.** Use `<Banner tone="warning|error|information">` instead. Same shape, same Badge, same surface. The consolidated `Banner` also supports `onDismiss` for a close button. See [ADR-004](../../docs/adrs/ADR-004-component-bloat-guardrails.md) §3.

This page exists for migration reference only. New code should not import `AlertBanner`.

---

## Migration

```tsx
// Before
<AlertBanner variant="warning" title="Heads up" description="..." />

// After
<Banner tone="warning" title="Heads up" description="..." />
```

The `variant` prop maps 1:1 to `tone`:

- `variant="information"` → `tone="information"`
- `variant="warning"` → `tone="warning"`
- `variant="error"` → `tone="error"`
