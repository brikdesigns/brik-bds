# Alert banner

Contextual banner for important messages requiring user attention. Uses a secondary surface background with an icon-only Badge, title, description, and optional action.

## Overview

<Canvas of={Stories.Default} />

## Variants

Each variant includes a colored icon-only Badge that communicates severity at a glance.

### Information
General informational messages. Neutral badge with circle-info icon.
<Canvas of={Stories.Information} />

### Warning
Attention-requiring situations. Yellow badge with triangle icon.
<Canvas of={Stories.Warning} />

### Error
Error states and failures. Red badge with triangle-exclamation icon.
<Canvas of={Stories.Error} />

### All variants
Side-by-side comparison of all variants.
<Canvas of={Stories.AllVariants} />

## Variants

### Title only
Minimal alert without description.
<Canvas of={Stories.TitleOnly} />

<ArgTypes of={Stories} />

## Usage

```tsx
import { AlertBanner } from '@brik/bds';
import { Button } from '@brik/bds';

<AlertBanner
  variant="information"
  title="Update available"
  description="A new version is ready to install"
  action={<Button variant="primary" size="sm">Update now</Button>}
/>
```
