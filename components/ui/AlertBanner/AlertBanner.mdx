# Alert banner

Contextual banner for important messages requiring user attention. Uses a secondary surface background with an icon, title, description, and optional action.

## Usage

```tsx
import { AlertBanner } from '@brik/bds';
import { Button } from '@brik/bds';

<AlertBanner
  icon="info"
  title="Update available"
  description="A new version is ready to install"
  action={<Button variant="primary" size="sm">Update now</Button>}
/>
```
