# Password input

Text input with a visibility toggle for password fields. Extends TextInput with a show/hide button, inheriting all its props except `type` and `iconAfter`.

## Usage

```tsx
import { PasswordInput } from '@brik/bds';

// Basic password input
<PasswordInput label="Password" />

// With validation error
<PasswordInput
  label="Password"
  error="Password must be at least 8 characters"
/>

// With helper text
<PasswordInput
  label="New password"
  helperText="Use at least 8 characters with a mix of letters and numbers"
/>
```
