# ActivityTimeline

<ComponentLinks slug="activity-timeline" />

Vertical, chronological list of events — each row a leading icon node, label, optional detail, and timestamp, connected top-to-bottom. Pass the whole sequence as the `events` array; the first entry marks the origin.
