# NotificationList

<ComponentLinks slug="notification-list" />

Vertical list of notification rows, each a `NotificationItem` with a title, body, timestamp, and read/unread state. Pass the `notifications` array; unread rows render with emphasis, read rows recede.

### Empty

## Patterns

`NotificationPopover` — the interactive composition: a bell trigger with an unread badge, click-to-read, and mark-all-read. Stateful behavior the list's args can't express on their own.
