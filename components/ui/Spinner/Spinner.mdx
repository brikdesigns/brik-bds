# Spinner

Loading indicator with animated rotation. Theme-colored by default, adapts to current theme context. Available in small and large sizes.

## Usage

```tsx
import { Spinner } from '@brik/bds';

// Small spinner (default)
<Spinner size="sm" />

// Large spinner
<Spinner size="lg" />

// In a loading button
<button disabled>
  <Spinner size="sm" style={{ borderTopColor: 'white' }} />
  Loading...
</button>

// Centered in container
<div style={{ display: 'flex', justifyContent: 'center', alignItems: 'center', minHeight: '200px' }}>
  <Spinner size="lg" />
</div>
```
