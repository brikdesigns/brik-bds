# Spinner

Circular loading indicator with animated rotation. Theme-colored by default, adapts to current theme context.

---

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Sizes

<Canvas of={Stories.Small} />
<Canvas of={Stories.Large} />

- **`sm`** — 16x16px, inline and button loading states
- **`lg`** — 48x48px, container and full-page loading

## Usage

```tsx
import { Spinner } from '@brik/bds';

// Small spinner (default)
<Spinner size="sm" />

// Large spinner
<Spinner size="lg" />

// Custom colors for dark backgrounds
<Spinner size="sm" style={{ borderColor: 'rgba(255,255,255,0.3)', borderTopColor: 'white' }} />
```
