# Spinner

Circular loading indicator with animated rotation. Theme-colored by default, adapts to current theme context.

## Variants

Size comparison: small (16px) for inline use, large (48px) for container loading.

<Canvas of={Stories.Variants} />

| Size | Dimensions | Use for |
|------|-----------|---------|
| **sm** | 16 x 16px | Inline, button loading states |
| **lg** | 48 x 48px | Container, full-page loading |

## Usage

```tsx
import { Spinner } from '@brik/bds';

// Small spinner (default)
<Spinner size="sm" />

// Large spinner
<Spinner size="lg" />

// Custom colors for dark backgrounds
<Spinner size="sm" style={{ borderColor: 'rgba(255,255,255,0.3)', borderTopColor: 'white' }} />
```
