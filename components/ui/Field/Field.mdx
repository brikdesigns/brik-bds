# Field

Label + value pair for read-mode display inside a Sheet. One API covers text, tags, URLs, bullet lists, and empty states.

This is the most important primitive in the sheet body set — it replaces four separate patterns (`FieldPair`, `TextValue`, `SmartTextValue`, raw label/value markup) that were drifting across the portal.

## Layouts

- **Stacked** (default) — label above value, full-width value.
- **Inline** — label left, value right. Use for compact stat rows ("Status · Active", "Scraped · 8").

## Empty states

When `children` is null, undefined, or empty string, `Field` renders an inline muted "Not set" string by default.

<Canvas of={Stories.Empty} />
<Canvas of={Stories.CustomEmpty} />

- Pass a string via `empty="..."` to customize the message per-field.
- The default "Not set" covers the generic case.

## Section-level empty

When an entire section has no data, pass an `<EmptyState>` into `empty`. Use the full bordered treatment sparingly — one per section at most.

<Canvas of={Stories.EmptyWithComponent} />

## Value-type recipes

`children` accepts any ReactNode — text, tags, links, lists, or compound content.

<Canvas of={Stories.ValueAsTagList} />
<Canvas of={Stories.ValueAsLink} />

Rules:

- **Don't style the value inline.** If a value needs tag treatment, use `<Tag>`. If it needs a link, use `<a>` (or `<TextLink>`). If it needs a list, use `<BulletList>`.
- **Don't nest Fields.** A value is a value — if you need sub-structure, it belongs in a new `SheetSection`.

## Usage

```tsx
import { Field, FieldGrid, TagGroup, Tag, BulletList } from '@brikdesigns/bds';

// Single stacked field
<Field label="Status">Active</Field>

// Empty with custom message
<Field label="Parent organization" empty="Independent" />

// Inline layout — compact stat row
<Field layout="inline" label="Scraped">8</Field>

// Value with tags
<Field label="Services">
  <TagGroup>
    <Tag size="sm">Cosmetic</Tag>
    <Tag size="sm">Implants</Tag>
  </TagGroup>
</Field>

// Value with a bullet list
<Field label="Anti-messages">
  <BulletList items={['No price-first', 'No corporate-clinic']} />
</Field>

// Inside a FieldGrid
<FieldGrid columns={2}>
  <Field label="Owner">Nick Stanerson</Field>
  <Field label="Location">Thompson Station, TN</Field>
</FieldGrid>
```
