# Tooltip

Contextual information displayed on hover or focus. Helps provide additional context without cluttering the interface. Supports four placement positions.

## Placement Options

### Top
Tooltip appears above the trigger element (default).

### Bottom

## Usage

```tsx
import { Tooltip } from '@brik/bds';

// Basic tooltip
<Tooltip content="This is helpful information">
  <button>Hover me</button>
</Tooltip>

// With placement
<Tooltip content="Appears below" placement="bottom">
  <button>Hover me</button>
</Tooltip>

// Icon button with tooltip
<Tooltip content="Edit item">
  <button aria-label="Edit">✏️</button>
</Tooltip>

// Help text
<div>
  <label>Username</label>
  <Tooltip content="Must be 3-20 characters" placement="right">
    <span>?</span>
  </Tooltip>
</div>
```
