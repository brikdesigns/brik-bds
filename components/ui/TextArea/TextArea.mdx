# TextArea

Multi-line text input with configurable rows, resize behavior, and placeholder support. Use for longer-form text entry like comments, descriptions, and messages.

<Canvas of={Stories.Default} />

## Variants

### With Default Value

<Canvas of={Stories.WithDefaultValue} />

### Disabled
Prevent interaction with the textarea.
<Canvas of={Stories.Disabled} />

### No Resize
Lock the textarea dimensions to prevent user resizing.
<Canvas of={Stories.NoResize} />

### Many Rows
Increase visible rows for longer content.
<Canvas of={Stories.ManyRows} />

## Usage

```tsx
import { TextArea } from '@brik/bds';

<TextArea
  placeholder="Enter your text here..."
  rows={4}
  resize="vertical"
  defaultValue="Pre-filled content"
/>
```
