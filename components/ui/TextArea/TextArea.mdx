# Text area

Multi-line text input with configurable rows, resize behavior, and size variants. Use for longer-form text entry like comments, descriptions, and messages.

## Variants

Sizes, states, and resize modes.

<Canvas of={Stories.Variants} />

| Size | Description |
|------|-------------|
| **sm** | 14px text — compact forms |
| **md** | 16px text — default |
| **lg** | 18px text — prominent inputs |

| Resize | Behavior |
|--------|----------|
| **vertical** | Default — height only |
| **none** | Fixed dimensions |
| **both** | Width and height |
| **horizontal** | Width only |

## Usage

```tsx
import { TextArea } from '@brik/bds';

// Basic textarea
<TextArea placeholder="Enter your text here..." rows={4} />

// With label and helper
<TextArea
  label="Notes"
  placeholder="Enter your message..."
  helperText="Maximum 500 characters"
  rows={4}
  fullWidth
/>

// Error state
<TextArea label="Description" error="This field is required" rows={3} fullWidth />

// No resize
<TextArea placeholder="Fixed size" rows={4} resize="none" />
```
