# Card control

Settings control card with badge, title, description, and an action element. Used for toggles, configuration options, and status displays.

## Action alignment

Use `actionAlign="top"` when the description wraps to multiple lines — the CTA anchors to the upper-right corner instead of drifting to the vertical center of the card.

## Usage

```tsx
import { CardControl, Badge, Switch, Button } from '@brik/bds';

<CardControl
  title="Notifications"
  description="Receive email notifications for updates"
  badge={<Badge size="xs" status="positive" icon={<CheckIcon />} />}
  action={<Switch checked={enabled} onChange={handleChange} />}
/>
```
