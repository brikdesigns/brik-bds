# Meter

<ComponentLinks slug="meter" />

Bounded-value indicator for completion counters, scores, and capacity displays. Pass `value` and `max`; the `status` prop drives color tokens (`positive`, `warning`, `error`, `neutral`).

## Default

<Canvas of={Stories.Default} />

## Variants

### Custom fill color

`status` drives the fill color through system tokens — toggle it (and `size`, `valueSuffix`, `label`) in Controls. `fillColor` is the escape hatch that overrides the status color for per-category or per-department palettes.
