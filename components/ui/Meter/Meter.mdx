# Meter

Visual gauge for displaying a value within a known range. Use for scores, completion rates, performance metrics, and health indicators.

## Usage

```tsx
import { Meter } from '@brik/bds';

// Basic meter
<Meter value={82} max={100} status="positive" label="Score" />

// With custom formatter
<Meter
  value={0.73}
  max={1}
  status="warning"
  label="Completion"
  formatter={(v) => `${Math.round(v * 100)}%`}
/>

// Small size, label below
<Meter value={45} max={100} size="sm" labelPosition="below" />
```
