# Skeleton

Content placeholder with shimmer animation for loading states. Maintains layout and reduces perceived load time.

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Variants

Three shape variants: text, circular, and rectangular.

<Canvas of={Stories.Variants} />

| Variant | Default size | Use for |
|---------|-------------|---------|
| **text** | 100% x 1em | Text line placeholders |
| **circular** | 40px | Avatar, icon placeholders |
| **rectangular** | 100% x 140px | Image, card, section placeholders |

## Patterns

Real-world loading states: article cards and comment lists.

<Canvas of={Stories.Patterns} />

<ArgTypes of={Stories} />

## Usage

```tsx
import { Skeleton } from '@brik/bds';

// Text line
<Skeleton variant="text" width="200px" />

// Avatar placeholder
<Skeleton variant="circular" width={48} height={48} />

// Image placeholder
<Skeleton variant="rectangular" width="300px" height="200px" />
```
