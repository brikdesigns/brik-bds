# Skeleton

Content placeholder with shimmer animation for loading states. Helps maintain layout and reduce perceived load time by showing where content will appear.

<Canvas of={Stories.Text} />

## Variants

### Text
Single-line text placeholder with automatic height based on font size.

### Circular
Round placeholder for avatars and profile images.

## Usage

```tsx
import { Skeleton } from '@brik/bds';

// Text line
<Skeleton variant="text" width="200px" />

// Avatar placeholder
<Skeleton variant="circular" width={48} height={48} />

// Image placeholder
<Skeleton variant="rectangular" width="300px" height="200px" />

// Card skeleton
<div>
  <Skeleton variant="rectangular" width="100%" height="160px" />
  <Skeleton variant="text" width="80%" />
  <Skeleton variant="text" width="100%" />
  <Skeleton variant="text" width="60%" />
</div>
```
