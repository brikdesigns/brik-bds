# Badge

Status indicators and labels for displaying contextual information. Badges use semantic colors to communicate status, priority, or category at a glance.

## Playground

Use the Controls panel to explore all props interactively.

<Canvas of={Stories.Playground} />

## Variants

Seven status variants across four sizes. The `xs` size renders a compact icon-only badge.

<Canvas of={Stories.Variants} />

| Status | Use for |
|--------|---------|
| **default** | General-purpose label. No specific semantic. |
| **positive** | Success states, completed actions. |
| **warning** | Cautionary states, pending actions. |
| **error** | Error states, failed actions, critical issues. |
| **info** | Informational states, neutral notifications. |
| **progress** | In-progress states, active processes. |
| **neutral** | Inactive states, disabled items. |

## Usage

```tsx
import { Badge } from '@brik/bds';

// Basic badge
<Badge>New</Badge>

// With status
<Badge status="positive">Success</Badge>
<Badge status="warning">Pending</Badge>
<Badge status="error">Failed</Badge>

// With size
<Badge size="sm">Small</Badge>
<Badge size="lg">Large</Badge>

// With icon
<Badge status="positive" icon={<CheckIcon />}>
  Complete
</Badge>
```
