# Snackbar

Brief notification bar that appears at the edge of the screen via portal. Auto-dismisses after a configurable duration. Use for non-critical feedback like "Item saved" or "Connection restored."

## Playground

Use the Controls panel to explore all props interactively.

## Variants

Status variants control the visual appearance.

<Canvas of={Stories.Variants} />

| Variant | Background | Text |
|---------|-----------|------|
| **default** | Black | Inverse |
| **success** | System green | On-color dark |
| **error** | System red | On-color dark |
| **warning** | System orange | On-color light |

## Usage

```tsx
import { Snackbar } from '@brik/bds';
import { useState } from 'react';

const [isOpen, setIsOpen] = useState(false);

<Snackbar
  isOpen={isOpen}
  onClose={() => setIsOpen(false)}
  message="Changes saved"
  variant="success"
  position="bottom"
  duration={3000}
/>
```
